## Description

NestJS Clean Architecture, Command-Query Responsibility Segregation, Domain Driven Design, Event Driven Architecture and Eventual Consistency in one repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
